# FAQ:
## Q: How i run it?
### R: Windows:
```
Double click the .jar or use the following command:
java -jar jmplayer_v<version>.jar
```
### Linux:
```sh
java -jar jmplayer_v<version>.jar
```
### macOS:
```bash
java -jar jmplayer_v<version>.jar
```
